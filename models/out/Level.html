Class: Level

Level(backgroundObjects, clouds, enemies, endboss, bottles, coins)

Represents a level in the game. A level has background objects, clouds, enemies, an end boss, bottles, coins, and a level end position.

Constructor

new Level(backgroundObjects, clouds, enemies, endboss, bottles, coins)

Creates a new level.
Parameters:
Name Type Description
backgroundObjects Array.<BackgroundObject> The background objects in the level.
clouds Array.<Cloud> The clouds in the level.
enemies Array.<Enemy> The enemies in the level.
endboss Endboss The end boss of the level.
bottles Array.<Bottle> The bottles in the level.
coins Array.<Coin> The coins in the level.
