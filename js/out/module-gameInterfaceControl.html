Module: gameInterfaceControl

This module manages the game control interface. It provides functions to setup the control interface, handle clicks on the control interface, toggle the visibility of the control interface, handle game pause or resume, close the control interface, and add or remove custom event listeners.

(inner) closeControlInterface()

Closes the control interface. It gets the overlay element and hides the control interface and the overlay if the control interface is visible. If the game has started, it also resumes the game and adds custom event listeners.
